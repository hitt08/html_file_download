## RUN
```
bash download_file.sh [OPTION]

OPTIONS
  -p, --pattern   File Pattern, Default: mp4
  -u|--url        Web URL
  -o|--outname    Output File Name, Default: Same as web
  -f|--file       File with list of WebURL[,OutputFileName] (separated in each line)
  ```
  ## File Format
  ```
  URL1
  URL2
  URL3
  ```
  OR
  ```
  URL1,FILE1
  URL2,FILE2
  URL3,FILE3
  ```
